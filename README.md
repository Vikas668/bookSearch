# Book Search App

Welcome to the Book Search App! This application allows users to search for books using the Open Library API and maintain a personal bookshelf.

## Setup

To set up the project locally, follow these steps:

1. **Clone the Repository**: 
   ```sh
   git clone https://github.com/your-username/book-search-app.git



2. Navigate to the Project Directory:
cd book-search-app

3. Install Dependencies:
npm install

4. Running the Application :
Once you've set up the project, you can run the application locally:
i.) Start the Development Server:
npm start
ii.) Open the Browser:
Visit http://localhost:3000 in your web browser to view the application.

**Usage**

i.) Book Search Page:
Use the search bar to type in the name of a book and view search results.
Click on "Add to Bookshelf" to add a book to your personal bookshelf.
ii.) Personal Bookshelf Page:
View and manage your personal bookshelf.
Books added to the bookshelf are stored locally in the browser.
